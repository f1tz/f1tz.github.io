网易云课堂加密视频H5播放方式，限MacOS

发表于 2018-08-06

1、使用Safari浏览器，切换UA为iPhone //必须使用Safari浏览器，Chrome切换UA也不行

2、使用移动端地址打开课程，把URL里的learn替换为course就行了

3、现在就已经可以通过HTML5播放了，视频加速可以手工用控制台，设置video标签的playbackRate属性。

4、或者使用我的油猴脚本，虽然写法比较粗糙，基本够用，比手动控制台要快一些。

// ==UserScript==
// @name         网易H5视频加速
// @namespace    http://tampermonkey.net/
// @version      0.1
// @description  lalala
// @author       F1tz
// @match        http://mooc.study.163.com/course/*
// @grant        none
// ==/UserScript==

(function() {
    'use strict';
    var selectEL = document.createElement('select');
    selectEL.setAttribute('id','rateSelect');
    var option1 = document.createElement('option');
    option1.text = '1倍速';
    option1.value = '1';
    var option2 = document.createElement('option');
    option2.text = '1.25倍速';
    option2.value = '1.25';
    var option3 = document.createElement('option');
    option3.text = '1.5倍速';
    option3.value = '1.5';
    var option4 = document.createElement('option');
    option4.text = '2倍速';
    option4.value = '2';
    selectEL.add(option1,null);
    selectEL.add(option2,null);
    selectEL.add(option3,null);
    selectEL.add(option4,null);
    document.getElementById('j-courseIntroHead').appendChild(selectEL);
    var targetVid = document.getElementsByTagName('video');
    document.getElementById('rateSelect').addEventListener("change",function(){
        targetVid[0].playbackRate = this.value;
    });
})();

GitHub部署Hexo问题总结

发表于 2017-03-06

之前有看过好多网友都在GitHub有搭建过博客，于是今天抽空自己也部署了一个Hexo在自己的GitHub上，把遇到的一些问题总结一下，避免以后犯错。以下演示基于MacOS 10.11.5环境。

声明下文内的 “{username} = 你的GitHub登录名”

GitHub端

1、配置SSH Keys

终端里生成RSA Key

sudo ssh-keygen -t rsa -C "{username}@github.com"

在Mac环境下，这里一定要使用root权限生成SSHKey，因为后面部署会需要root权限。
之前遇到个一个情况，第一次可以使用Hexo部署，没有报错，但之后就不行了，后来基于root重新生成的Key才解决。
